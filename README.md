# OpenBox Website 📦
Welcome to the **OpenBox Builder Hub** — a global community and digital platform for developers, engineers, and creators. OpenBox is designed to help builders connect, collaborate on projects, and level up their engineering skills through an active and structured community.
## 🚀 Features
- **Dynamic Documentation:** Comprehensive system docs detailing server roles, community guidelines, and bot commands.
- **Categorized Roles Structure:** Granular progression system that encompasses technical divisions, academic standing, and supporter tiers.
- **Events & Runway:** Active calendar scheduling for all upcoming tech talks, networking sessions, and collaborative builds.
- **Support & Moderation System:** Integrated appeals and assistance portal for maintaining community safety.
## 🛠 Tech Stack
- **Frontend Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Routing:** [@tanstack/react-router](https://tanstack.com/router/latest)
- **Language:** TypeScript
- **Styling:** Custom CSS (with theming support)
## 📦 Getting Started
### Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/VSBro13/openbox-builder-hub.git
   ```
2. Navigate to the project directory:
   ```bash
   cd openbox-builder-hub
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
### Running the Development Server
To start the local development server, run:
```bash
npm run dev
```
The application will typically be accessible at `http://localhost:8080`.
## 🎨 Theming
OpenBox supports multiple custom themes! Themes can be toggled and are saved locally to your browser. The currently supported themes are:
- `black` (Default)
- `deep-blue`
- `violet`
- `slate`
- `teal`
